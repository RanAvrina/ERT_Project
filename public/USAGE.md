# 🚀 הוראות הפעלה - ERT

## איך להתחיל?

### 1. פתח את העמוד הראשי
```
פתח ב-Browser: public/index.html
```

כך גם תוכל לפתוח:
```bash
# Windows
start public\index.html

# Mac
open public/index.html

# Linux
xdg-open public/index.html
```

### 2. התחבר או השמרה ההתחברות

דף התחברות יופיע כאשר אתה מפתח את האפליקציה. אתה יכול:
- להתחבר עם אימייל וסיסמה
- ללחוץ "הרשמה" כדי ליצור חשבון חדש
- ללחוץ "שכחתי סיסמה" אם שכחת

### 3. בחר דף מהתפריט

לאחר התחברות, תוכל לנווט בקלות בין הדפים שונים:
- 📊 דשבורד - עמוד ראשי עם סטטיסטיקות
- 🏠 מידע דירה - פרטי הדירה
- 👥 שותפים - ניהול שותפי הדירה
- 💰 הוצאות - עקוב אחרי הוצאות משותפות
- ✓ משימות - רשימת משימות
- 🛒 קניות - קניות משותפות
- 💳 תשלומים - ניהול תשלומים
- 🎫 כרטיסים - דיווחים בעיות

## 📋 ניווט וספרתי

  - צפה בכלל ההוצאות החודשיות
  - צפה בשיתופים פתוחים

  - צפה בכרטיסים פתוחים

- **מה תוכל לעשות:**
  - צפה בכתובת הדירה
  - צפה בשטח הדירה
  - צפה בשכירות
  - ערכן פרטים אם אתה משכיר

  - צפה בשותפים בדירה

- **מה תוכל לעשות:**
  - הוסף משימה חדשה

### קניות
- **מטרה**: ניהול רשימת קניות משותפת
- **מה תוכל לעשות:**
  - הוסף פריט לרשימה
  - סמנו כברצוי כשקנית
  - ראה כמות של כל פריט

### תשלומים

  - רשום תשלום בין שותפים

### כרטיסים

## 💾 אחסון נתונים

כל הנתונים שלך נשמרים באופן השגחת ב-localStorage של הדפדפן. זה אומר:
- ✅ הנתונים נשארים גם אחרי סגירת הדפדפן
- ✅ אין צורך בחיבור אינטרנט
- ✅ הנתונים פרטיים לך בלבד

**הערה:** אם תמחק את נתוני הדפדפן, כל הנתונים מחקו.

## ⚠️ עצות חשובות

### 1. הLogin מתוך בפעם הראשונה
- כל משתמש יכול להתחבר עם כל אימייל וסיסמה
- בפעם הבאה, השתמש באותם פרטים

### 2. שמור את האימייל שלך
- בדף התחברות יש אפשרות "זכור אותי"
- זה שומר את האימייל שלך לאחר התחברויות הבאות

### 3. קוד הזמנה לשותפים
- דף השותפים מאפשר ליצור קוד הזמנה
- בעוד שותף יכול להשתמש בקוד כדי להצטרף לדירה

### 4. פילוח שליטה על הנתונים
- אתה יכול לערוך פרטים של הדירה
- אתה יכול למחוק פריטים מהרשימות

### 5. תמיד בדוק את הסטטוס
- כרטיסים כוללים סטטוס (פתוח, בעבודה, סגור)
- הוצאות מקבוצות לקטגוריות

## 🔧 Troubleshooting

### בעיה: הדף לא טוען
**פתרון:**
1. רענן את הדפדפן (F5)
2. בדוק שאתה פותח את `public/index.html`
3. בדוק את קונסול התעודות (F12) לשגיאות

### בעיה: הנתונים לא נשמרו
**פתרון:**
1. בדוק שה-localStorage מופעל בדפדפן
2. בדוק שיש מקום זכרון זמין
3. סגור והפתח את הדפדפן שוב

### בעיה: עמוד מסוים לא עובד
**פתרון:**
1. רענן את הדף
2. נקה את localStorage (כאופציה אחרונה)
3. בדוק בעקבות הבדיקה (F12)

### בעיה: CSS -ים לא מטופלים
**פתרון:**
1. בדוק שגרון מסך מלא (F11)
2. בדוק בעקבות בדיקה אם קובץ ה-CSS טוען
3. רענן את הדף
