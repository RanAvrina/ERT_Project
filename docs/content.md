# Project Context

## Project Name
ERT - Shared Apartment Management System

## Goal
A responsive mobile-first web app for shared apartment management.

## Main Modules
- Authentication and user management
- Dashboard
- Expenses and debt calculation
- Payments between roommates
- Tasks
- Shopping list
- Maintenance / landlord tickets

## Design Direction
- Clean, modern, minimal UI
- Responsive and mobile-first
- Hebrew RTL interface
- Primary color: #2EC4B6
- Light background: #FFFDF7
- White cards
- Visual dashboard

## Current Stage
We are not connecting to a database yet.
First, build the frontend skeleton with mock data only.

## Important Rules
- Do not invent features that are not in the documents
- Follow the docs folder as the source of truth
- Keep the app mobile-first and RTL
- Build reusable components
